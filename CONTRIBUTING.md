# Contributing

Thanks for helping improve Signal Search.

## Getting started
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## Development workflow
- Create focused, atomic commits with Conventional Commit messages.
- Run `npm run lint` and `npm test` before opening a PR.

## Code style
- TypeScript only.
- Prefer small, composable components.
- Keep UI copy concise and actionable.

## Reporting issues
Please include steps to reproduce, expected behavior, and screenshots where possible.
